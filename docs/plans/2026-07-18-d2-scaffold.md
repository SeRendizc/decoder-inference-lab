# D2 Model Lab Scaffold Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use executing-plans to implement this plan task-by-task. Steps use checkbox syntax for tracking.

**Goal:** 建立不包含 attention 实现的 Model Lab 外围骨架，使配置、byte tokenizer、next-token 样本与环境验证可测试运行。

**Architecture:** 源码采用 `src/` layout。`config.py` 只负责模型超参数与 YAML 校验，`data.py` 只负责 UTF-8 byte tokenizer 和 next-token 样本构造。核心模型目录只建立 ownership 说明，不生成 attention 算法。

**Tech Stack:** Python 3.10、PyTorch 2.12.1、PyYAML、pytest、Ruff。
